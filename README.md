## 📌 학습 목차

### [Chapter 2: 차 상태 예측 분류 모델 (car_evaluation)](./char2/)
* **개요:** 차량의 6가지 데이터(나이, 주행거리 등)를 바탕으로 4가지 차량 상태를 예측하는 다중 분류 모델
* **핵심 학습 내용:**
  * `CrossEntropyLoss`와 `Adam` 옵티마이저 활용
  * `with torch.no_grad()`를 통한 평가 모드 제어 및 `np.argmax` 예측값 변환
  * 성능 평가 지표 분석 (정확도 75% 달성 및 다수 클래스 편향 문제 확인)
* **결과 요약:** * 전체 정확도는 75%이지만, 특정 클래스(1번)에 대한 재현율(Recall)이 낮아 데이터 불균형 문제를 인지함. 향후 모델 구조 개선 필요.
